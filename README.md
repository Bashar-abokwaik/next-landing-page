# NovaStyle Landing Page

Modern landing page built with Next.js to showcase the NovaStyle brand and products.

The landing page focuses on delivering a fast, responsive, and visually appealing experience while highlighting featured products, best sellers, newsletter subscription, and company information.

---

## Live Demo

https://novastyle-home.web.app

---

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- React Query
- Framer Motion

---

## Features

- Hero Section
- Best Sellers
- About Section
- Newsletter Subscription
- Responsive Design
- Dark Mode
- Smooth Animations

---

## Installation

```bash
git clone <repository>

npm install

npm run dev
```
